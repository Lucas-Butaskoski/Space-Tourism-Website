## Visão geral

### O desafio

Os usuários devem ser capazes de:

- Visualizar o layout ideal para cada página do site de acordo com o tamanho da tela do dispositivo
- Ver os estados de hover nos elementos interativos
- Navegar entre as páginas do site
- Alternar entre as abas de **Destination**, **Crew** e **Technology** para visualizar diferentes conteúdos

## Meu processo

### Construído com

- HTML5 semântico
- CSS3
- Flexbox
- CSS Grid
- JavaScript (Vanilla)
- Layout responsivo (desktop, tablet e mobile)
- Menu hambúrguer para dispositivos móveis

### O que aprendi

Neste projeto, consegui reforçar conceitos importantes do desenvolvimento front-end, como:

- Estruturação de páginas utilizando HTML semântico
- Criação de layouts responsivos com Flexbox e Grid
- Manipulação do DOM com JavaScript para trocar conteúdos dinamicamente
- Implementação de menus responsivos para mobile
- Uso de animações simples para melhorar a experiência do usuário
- Organização de código CSS para projetos com múltiplas páginas

Esse desafio me ajudou a ganhar mais confiança na construção de interfaces e na leitura de layouts a partir de designs prontos.

### Desenvolvimento contínuo

Para melhorar este projeto no futuro, pretendo:

- Utilizar um arquivo `data.json` com `fetch` para centralizar os dados
- Melhorar a acessibilidade do site
- Refatorar partes do código para deixá-lo mais organizado
- Evoluir o projeto usando frameworks como React futuramente
